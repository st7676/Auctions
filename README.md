Auctions App
מערכת ניהול יומן אישית עם משתמשים ואירועים (Events). הפרויקט בנוי ב־Angular (Frontend) ו־ASP.NET Core Web API (Backend).
ומסד הנתונים: קובץ sqlite
בשרת ה.net: הוספת מכירה פומבית, הצגת מכירה, הצגת כל המכירות, הצגת הההצעות של מכירה מסוימת
בangular: בדף הראשי מוצגות כל המכירות, עם כפתור: כניסה למכירה,
בפנים ניתן לצפות בהצעה הרלוונטית האחרונה למכירה זו וכן להציע הצעה חדשה,
מתקיימות בדיקות בסרוויס: הצעה גבוהה מההצעה הנוכחית, סטטוס המכירה תקף התאריך לא עבר.
על מנת לאפשר שימוש בו זמנית של מספר משתתפים, יש להציג נתונים עדכניים על הדפדפן, להציג את ההצעה האחרונה (לדוג') ברגע בה היא עולה למסד הנתונים ולא בעת רענון האתר,
